1. **개념 먼저** — 무작정 정답 암기 금지, 배경 지식부터 이해
2. **메커니즘 시각화** — ASCII 다이어그램 / 흐름도로 구조 파악
3. **단계별 트레이싱** — 코드·계산 문제는 한 줄씩 직접 추적
4. **비교 표** — 헷갈리는 유사 개념은 항상 나란히 비교
5. **핵심 암기 포인트** — 각 문서 하단에 시험 직전 정리 제공
