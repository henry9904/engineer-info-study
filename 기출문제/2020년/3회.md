## Q04. 분기 커버리지 경로

### 지문 키워드
> 제어 흐름 그래프의 분기 커버리지를 7단계·6단계 경로로 나열

### 개념 — 분기(Branch) 커버리지

> 각 분기(간선)를 최소 1회 실행하는 경로들의 조합. 모든 간선이 커버되도록 두 경로 제시.

### ✅ 정답 (그래프 구조 기준)
```
① → ② → ③ → ④ → ⑤ → ⑥ → ①  (7단계)
① → ② → ④ → ⑤ → ⑥ → ⑦        (6단계)
```
> ⚠️ 두 경로를 합쳐 모든 분기(간선)가 실행되면 정답.

## Q07. SQL — GROUP BY + HAVING + AS

### 지문 키워드
> 과목별 평균 90 이상인 과목이름·최소·최대점수, **WHERE 금지, GROUP BY·HAVING·AS 사용**

### 단계별 트레이싱
```
1. GROUP BY 과목이름 → 데이터베이스(95,99), 자료구조(80,88,79)
2. HAVING AVG(점수)>=90 → 데이터베이스(97)✅, 자료구조(82.3)✗
3. SELECT 과목이름, MIN=95, MAX=99
```

### ✅ 정답
```sql
SELECT 과목이름, MIN(점수) AS 최소점수, MAX(점수) AS 최대점수
FROM 성적
GROUP BY 과목이름
HAVING AVG(점수) >= 90;
```

## Q16. C — 곱셈 누적 (초기값 0)

### 문제
```c
int i = 0, c = 0;
while (i < 10) {
    i++;
    c *= i;      // c = c * i
}
printf("%d", c);
```

### 핵심 개념 — 0에 무엇을 곱해도 0

> ⚠️ **핵심 함정**: `c`의 초기값이 **0**. `c *= i`는 0×i = 0 → 아무리 곱해도 계속 0.

### ✅ 정답
```
0
```

## Q18. Java — continue (짝수 합)

### 문제
```java
int i = 0, sum = 0;
while (i < 10) {
    i++;
    if (i % 2 == 1) continue;   // 홀수면 건너뜀
    sum += i;                   // 짝수만 더함
}
System.out.print(sum);
```

### 핵심 개념 — continue는 이후 문장 건너뛰고 다음 반복

> 홀수(i%2==1)면 `continue` → sum 더하기 스킵. 짝수만 누적.

### 단계별 트레이싱

```
짝수만: 2 + 4 + 6 + 8 + 10 = 30
```

### ✅ 정답
```
30
```

## Q19. Java — 오버로딩 + 상속 getName

### 문제
```java
abstract class Vehicle {
    String name;
    public Vehicle(String val) { this.name = val; }
    public String getName() { return "Vehicle name : " + name; }
}
class Car extends Vehicle {
    private String name;
    public Car(String val) { name = super.name = val; }
    public String getName(String val)  { return "Car name : " + val; }   // 오버로드
    public String getName(byte val[])  { return "Car name : " + val; }   // 오버로드
}
Vehicle obj = new Car("Spark");
System.out.println(obj.getName());   // 무인자!
```

### 핵심 개념 — 오버라이딩 아닌 오버로딩

> ⚠️ **핵심 함정**: Car의 `getName(String)`, `getName(byte[])`은 **매개변수가 달라 오버로딩**(별개 메서드) — 부모의 무인자 `getName()`을 재정의한 게 아님. 따라서 `obj.getName()`(무인자)은 Car에 없어 **Vehicle.getName()** 호출. name = "Spark".

### ✅ 정답
```
Vehicle name : Spark
```
