## Q01. 반정규화 (Denormalization)

## Q17. LOC 개발 기간

### 지문 키워드
> 총 30,000라인, 프로그래머 5명, 월 생산성 300라인/인

### 핵심 개념 — LOC 개발 기간 공식

```
개발 기간(월) = 총 LOC ÷ (인원 × 1인당 월 생산성)
              = 30000 ÷ (5 × 300)
              = 30000 ÷ 1500
              = 20개월
```
> (또는 노력=30000/300=100인월, 기간=100/5=20개월)

### ✅ 정답
```
30000 / (5 × 300) = 20개월
```

## Q18. C — 버블 정렬

### 문제
```c
int a[] = {95, 75, 85, 100, 50};
for (i=0; i<n-1; i++)
```

```
→ 정렬 → {50, 75, 85, 95, 100}
```

## Q20. Java — switch fall-through

### 문제
```java
int i = 3, c = 1;
switch (i) {
    case 1: c++;
    case 2: c += 3;
    case 3: c = 0;
    case 4: c += 3;
    case 5: c -= 10;
    default: c--;
}
System.out.print(c);
```

### 핵심 개념 — break 없음 → case 3부터 끝까지 실행

> ⚠️ **핵심 함정**: break가 없어 i=3 → case 3부터 **아래 전부** 실행.

### 단계별 트레이싱

```
case 3: c = 0       → 0
case 4: c += 3      → 3
case 5: c -= 10     → -7
default: c--        → -8
```

### ✅ 정답
```
-8
```

## 📌 이 회차 핵심 암기

- **반정규화**: 성능 위해 중복 허용 / **살충제 패러독스**: 반복 테스트는 새 결함 못 찾음
- **결합도 낮게(Low) + 응집도 높게(High)** = 좋은 설계
- **Fan-in**(들어옴) vs Fan-out(나감)
- **JSON**: 이름:값 쌍, XML 대체 / **성능 지표**: 처리량·응답시간·경과시간
- **Land Attack**: 출발지=목적지 IP → 자원 고갈
- **프로토콜 3요소**: 구문·의미·타이밍
- **HRN** = (대기+서비스)/서비스
- **ACID**: 원자성·일관성·격리성·지속성
- **물리 계층**: 비트 전송 (1계층)
- **LOC 기간** = 총LOC ÷ (인원×월생산성) (Q17)
- **switch break 없으면 fall-through** (Q20)
