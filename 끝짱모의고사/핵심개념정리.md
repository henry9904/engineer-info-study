## 1. 네트워크 / 통신

### OSI 7계층 & 장비 `[7회 Q15]` `[10회 Q10]`

```
7 응용 - 6 표현 - 5 세션 - 4 전송 - 3 네트워크 - 2 데이터링크 - 1 물리
암기: "아파세전네다물"
```

| 계층      | 데이터 단위 | 주소     | 장비        |
| ------- | ------ | ------ | --------- |
| 물리(1)   | 비트     | -      | 리피터, 허브   |
| 데이터링크(2) | 프레임    | MAC    | 브리지, 스위치  |
| 네트워크(3) | 패킷     | IP     | 라우터       |
| 전송(4)   | 세그먼트   | 포트     | -         |

- **패킷**(네트워크/IP) vs **프레임**(데이터링크/MAC) `[5회 Q9]`
- 리피터(신호 증폭) vs 스위치(MAC 기반 포트 전달) `[10회 Q10]`

### 프로토콜

| 프로토콜   | 계층/용도                     | 회차          |
| ------ | ------------------------- | ----------- |
| TCP    | 연결형, 신뢰성/순서 보장, 재전송        | `[4회 Q20]`  |
| UDP    | 비연결형, 실시간, 재전송 없음          | `[4회 Q20]`  |
| ICMP   | 오류 메시지, ping/traceroute    | `[3회 Q12]`  |
| IGMP   | 멀티캐스트 그룹 관리                | `[3회 Q12]`  |
| HDLC   | 데이터링크, 비트 스터핑, WAN         | `[2회 Q15]`  |
| CSMA/CD | 이더넷 매체 접근(충돌 감지)           | `[9회 Q1]`   |
| TELNET/SSH | 원격 접속(평문/암호화)            | `[9회 Q18]`  |
| L2TP   | VPN 터널링(+IPSec 결합)         | `[7회 Q1]`   |

### IP 클래스 `[7회 Q17]`

| 클래스 | 첫 옥텟   | 첫 비트  |
| --- | ------ | ----- |
| A  | 0~127  | 0     |
| B  | 128~191 | 10    |
| C  | 192~223 | 110   |

### 다중화 `[6회 Q3]`

- FDM(주파수 분할) / **TDM(시분할)**: 동기식(고정 슬롯, 낭비) vs 통계적(효율↑)

### 웹/데이터 포맷

- **REST**(HTTP 메서드+URI+JSON) vs **SOAP**(XML+WSDL, 기업용) `[9회 Q15]`
- SOAP/WSDL/UDDI, HTTP/HTTPS `[2회 Q5]`
- XML(사용자 정의 태그) / JSON(경량 키-값) / HTML(콘텐츠 표현) `[6회 Q10]`
- AJAX(비동기 부분 갱신) / 반응형 웹 `[1회 Q16]`

### 미들웨어 `[10회 Q9]`

- **RPC**: 원격 함수 로컬처럼 호출(동기)
- **MOM**: 메시지 큐 비동기(느슨한 결합)

## 2. 정보보안 / 암호

### 정보보안 3요소(CIA) & 인증 요소

- 기밀성 / 무결성 / **가용성**(Land Attack·DoS가 침해) `[3회 Q7]`
- 식별 → 인증(비번 확인) → 인가(권한) / 부인방지 `[2회 Q4]`

### 공격 유형

| 공격             | 특징                       | 회차          |
| -------------- | ------------------------ | ----------- |
| Land Attack    | 출발지 IP=목적지 IP(자기 응답, DoS) | `[3회 Q7]`   |
| 살라미            | 소액을 여러 번 조금씩              | `[1회 Q7]`   |
| 스턱스넷           | 기반시설(원전) 공격 웜             | `[1회 Q7]`   |
| 제로데이           | 패치 전 취약점 악용               | `[1회 Q7]`   |
| DNS 스푸핑 / 파밍   | DNS 위조 → 가짜 사이트           | `[7회 Q12]`  |
| DRDoS          | 반사 서버로 트래픽 증폭             | `[8회 Q13]`  |
| 중간자 공격(MITM)   | 통신 가로채 복호화·재암호화           | `[9회 Q12]`  |
| XSS / CSRF     | 스크립트 주입 / 위조 요청           | `[5회 Q13]`  |

- **능동적 공격**(변조·재전송·위장·DoS) vs **수동적 공격**(도청·트래픽 분석·감시) `[10회 Q7]`

### 암호 알고리즘 `[6회 Q19]` `[10회 Q1]`

- **RSA**: 비대칭키, 소인수분해, 키 교환/전자서명
- **AES**: 대칭키, NIST 표준, 블록 128비트
- 공개키 용도: **암호화=상대 공개키 / 전자서명=내 개인키** `[10회 Q1]`
- DB 암호화: **API 방식**(앱 수준) / **TDE**(DB 엔진 투명 암호화) `[10회 Q5]`

### 보안 모델 / 시스템

- 무결성 모델: 비바 / **클락-윌슨**(잘 정의된 트랜잭션·직무분리) / 만리장성 `[10회 Q3]`
- **IDS**(탐지·경고) / **IPS**(탐지+능동 차단) `[4회 Q17]`
- **제로 트러스트**: Never Trust, Always Verify `[8회 Q8]`
- 접근제어: **MAC**(강제) / **DAC**(소유자) / **RBAC**(역할) `[2회 Q18]`
- OTP / OAuth / SSO `[3회, 1회 Q14]`

## 3. 운영체제 / 시스템

### CPU 스케줄링

| 알고리즘 | 특징                     | 회차          |
| ---- | ---------------------- | ----------- |
| RR   | 시간 할당량(퀀텀) 순환, 기아 완화    | `[1회 Q12]` `[8회 Q7]` |
| SRT  | 남은 시간 최소 선점             | `[1회 Q12]`  |
| FCFS | 도착 순, 순서에 따라 평균 반환시간 변동 | `[5회 Q20]`  |

- 반환시간 = 완료 − 도착. SJF(짧은 것 먼저)가 평균 최소.
- **기아현상** ↔ **에이징**(대기 시 우선순위 상승) `[8회 Q7]`

### 메모리 / 페이지

- 가상주소 = VPN + 오프셋(페이지 크기 비트) `[2회 Q20]`
- 페이지 교체: **LRU**(최근성) / **LFU**(빈도, 동률 FIFO) `[4회 Q9]`
- **워킹 셋**(자주 참조 페이지 집합) ↔ **스래싱**(교체에만 매달림) `[4회 Q10]`
- 단편화: **내부**(고정 분할) / **외부**(가변 분할) / **압축**(Compaction) `[8회 Q20]`

### 프로세스 / 동기화

- **문맥 교환**: 상태 저장(PCB)+복원, 순수 오버헤드 `[3회 Q2]`
- **임계 영역** ↔ **상호 배제**(한 번에 하나만 진입) `[7회 Q13]`
- 교착상태 4조건: 상호배제·점유대기·비선점·환형대기 `[1회 Q13]`
- 교착 처리: 예방 / **회피(은행가 알고리즘)** / 탐지 / 회복 `[9회 Q11]`

### 시스템 운영 방식

- **다중 프로그래밍**(유휴 최소화) / **시분할**(짧은 응답) `[8회 Q18]`
- I/O 방식: 폴링 / 인터럽트 / 스풀링 `[5회 Q7]`
- 성능 지표: 반환시간 / 처리량 / 응답시간 `[6회 Q18]`

### 리눅스

- chmod(권한 변경) / umask(기본 차감) `[5회 Q17]`

## 5. 데이터베이스

### 키 & 무결성

- 후보키 = **유일성 + 최소성** `[6회 Q1]`
- 개체 무결성: PK는 **NULL 불가 + 중복 불가** `[8회 Q17]`
- 트랜잭션 ACID(원자성/일관성/격리성/지속성) `[3회 Q1]`
- 트랜잭션 상태: 활동 → 부분완료 → 완료 / 실패 → 철회 `[8회 Q2]`
- 회복: 즉시 갱신(redo+undo) / 지연 갱신(redo) `[2회 Q6]`

### SQL 집계 함정 (자주 나옴!)

- **COUNT(컬럼)·SUM·AVG는 NULL 제외**, COUNT(*)만 전체 `[9회 Q10]` `[10회 Q18]`
- COUNT(...) 결과는 **튜플 1개** 반환 `[3회 Q13]`
- WHERE(그룹화 전) vs HAVING(집계값 조건) `[3회 Q5]`
- SQL 실행 순서: FROM → WHERE → GROUP BY → HAVING → SELECT → ORDER BY `[7회 Q18]`

### SQL 문법

| 구문             | 회차          |
| -------------- | ----------- |
| GROUP BY + HAVING | `[3회 Q5]` `[6회 Q14]` |
| LEFT JOIN + COUNT(NULL) | `[2회 Q17]` `[8회 Q12]` |
| FULL OUTER JOIN | `[5회 Q6]`   |
| ANY / ALL 서브쿼리  | `[6회 Q8]`   |
| EXISTS 상관 서브쿼리  | `[4회 Q5]`   |
| 다중 중첩 서브쿼리      | `[10회 Q11]` |
| BETWEEN + 정렬    | `[7회 Q16]`  |
| GRANT/REVOKE (WITH GRANT OPTION/CASCADE) | `[1회 Q8]` |
| ALTER ADD/MODIFY | `[5회 Q14]`  |
| CREATE INDEX(복합) | `[9회 Q19]`  |
| 트리거(INSERT/UPDATE/DELETE) | `[2회 Q1]` |

### 관계대수

- σ(선택/행) / π(추출/열) / ∧(AND) `[4회 Q1]`
- **나눗셈(÷)**: S의 모든 값과 짝을 이루는 것 `[7회 Q9]`

- 정형 / 비정형 / NoSQL `[8회 Q6]`
- OLAP(다차원 분석) `[9회 Q3]`
- 색인 파일(키-주소 테이블) `[10회 Q16]`

## 6. 프로그래밍 공통 함정

### ⭐ Java 필드/메서드 바인딩 (거의 매회 출제!)

```
필드(field)      → 선언 타입 기준 (다형성 X, 은닉)
정적 메서드(static) → 선언 타입 기준
인스턴스 메서드    → 실제 객체 타입 기준 (오버라이딩 O)
```
- 생성 순서: **super() → 자식 필드 초기화 → 자식 생성자 본문**
- 생성자에서 오버라이딩 메서드 호출 시, **자식 필드가 아직 미초기화(null/0)**
  - `[3회 Q4]` `[6회 Q12]` `[9회 Q17]` `[1회 Q5]` `[5회 Q3]`

### Java 문자열 결합 `[9회 Q7]` `[8회 Q4]`

- `int + int + String` → 왼쪽부터: 산술 먼저 후 연결 (`5+6+"8"` = "118")
- `String + int + int` → 전부 연결 (`"8"+5+6` = "856")
- String ==(참조) vs equals(내용), intern()은 Pool 반환 `[3회 Q16]`

### Java 예외 `[2회 Q11]` `[7회 Q20]`

- **finally의 return이 catch의 throw/return을 덮어씀** (예외 소멸)

### C 포인터

- `int(*)[N]` 배열 포인터, 이중 포인터 `**`, 포인터 차 = 인덱스 차 `[10회 Q2]`
- 연결 리스트: next 연결 순서로 순회, value만 swap vs 연결 재배치 `[3회 Q14]` `[6회 Q4]`

## 8. 계산 문제 공식 모음

### 서브넷 `[1회 Q20]` `[3회 Q6]` `[9회 Q20]`

```
사용 가능 호스트 = 2^h − 2  (h=호스트 비트, 네트워크·브로드캐스트 제외)
서브넷 개수     = 2^s        (s=추가 서브넷 비트)
서브넷 마스크    = 32 − h 비트
서브넷 크기     = 2^h 주소
```
- 예: /24에서 호스트 40 이상 → 2^6−2=62 → h=6 → 마스크 /26, 서브넷 4개

### CPM(임계경로) `[3회 Q11]`

```
ES = 선행 EF의 최댓값,   EF = ES + 기간
LF = 후행 LS의 최솟값(없으면 종료일),  LS = LF − 기간
Slack = LS − ES = LF − EF
임계경로 = Slack 0인 경로(가장 긴 경로)
```

### 스케줄링 평균 반환시간 `[1회 Q12]` `[5회 Q20]`

```
반환시간 = 완료시간 − 도착시간
평균 반환시간 = Σ반환시간 / 프로세스 수
FCFS는 짧은 작업 먼저(SJF) 배치 시 평균 최소
```

### LOC 비용 `[6회 Q17]`

```
필요 인월(MM) = 총 LOC / 생산성(라인/인월)
개발비 = 인월 × 인건비
(관리비·QA비는 개발비의 %) — 인원수는 기간에만 영향(비용엔 함정)
```

### 가상주소 비트 `[2회 Q20]`

```
페이지 오프셋 비트 = log2(페이지 크기)
VPN 비트 = 전체 주소 비트 − 오프셋 비트
```
