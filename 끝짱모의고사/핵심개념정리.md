## 1. 네트워크 / 통신

### ⭐ OSI 7계층 & 장비/프로토콜

```
7 응용 - 6 표현 - 5 세션 - 4 전송 - 3 네트워크 - 2 데이터링크 - 1 물리
암기: "아파세전네다물"
```

| 계층 | 단위 | 주소 | 장비 | 대표 프로토콜 |
| --- | -- | -- | -- | -------- |
| 물리(1) | 비트 | - | 리피터, 허브 | - |
| 데이터링크(2) | 프레임 | MAC | 브리지, 스위치 | HDLC, PPP, Ethernet |
| 네트워크(3) | 패킷 | IP | 라우터 | IP, ICMP, IGMP, ARP |
| 전송(4) | 세그먼트 | 포트 | - | TCP, UDP |
| 응용(7) | 메시지 | - | - | HTTP, FTP, SMTP, SNMP, DHCP |

- 계층별 장비: 리피터(물리)/스위치(MAC)/라우터(IP) `[10회 Q10]`
- OSI 계층별 프로토콜 매칭 `[12회 Q3]` `[7회 Q15]`

### 프로토콜 총정리

| 프로토콜 | 용도 | 회차 |
| ---- | -- | -- |
| TCP / UDP | 연결·신뢰성 / 비연결·실시간 | `[4회 Q20]` |
| ICMP / IGMP | 오류·ping / 멀티캐스트 그룹 | `[3회 Q12]` `[14회 Q7]` |
| HDLC / X.25 | 비트 지향 데이터링크 / 가상회선 패킷교환 | `[2회 Q15]` `[15회 Q8]` |
| CSMA/CD | 이더넷 충돌 감지 | `[9회 Q1]` |
| FTP/TFTP/SFTP | 파일 전송(신뢰/간단/암호화) | `[17회 Q2]` |
| SNMP / DHCP | 장비 관리 / IP 자동할당 | `[17회 Q2]` `[14회 Q7]` |
| TELNET / SSH | 원격 접속(평문/암호화) | `[9회 Q18]` |
| L2TP / VPN | 터널링 / 가상사설망 | `[7회 Q1]` |
| POP3 / SMTP | 메일 수신 / 발송 | `[12회 Q5]` |

### ⭐ IP 주소 / 서브넷

- IP 클래스: A(0~127)/B(128~191)/C(192~223) `[7회 Q17]`
- NAT / PAT / NAT 테이블 (사설↔공인 변환) `[11회 Q7]`
- IPv4 vs IPv6: 헤더 고정/체크섬 제거/애니캐스트/IPSec 기본 `[12회 Q15]` `[8회 Q9]`
- IPv 전환: 터널링 / NAT-PT / 듀얼 스택 `[20회 Q1]`
- 단편화 / MTU (최대 전송 단위) `[16회 Q16]`

### 전송 방식 / 효율

- 유니캐스트(1:1) / 멀티캐스트(그룹) / 애니캐스트(가장 가까운 1) `[16회 Q8]`
- 통신 방향: 단방향 / 반이중(교대) / 전이중(동시) `[19회 Q1]`
- 다중화: FDM(주파수) / TDM(시분할) `[6회 Q3]`
- 피기배킹(ACK+데이터) / 슬라이딩 윈도우(연속 전송) `[20회 Q17]`
- ⭐ 프로토콜 3요소: **구문(Syntax) / 의미(Semantics) / 타이밍(Timing)** `[19회 Q7]`

### 웹 / 데이터 포맷

- REST(HTTP+URI+JSON) vs SOAP(XML+WSDL) `[9회 Q15]`
- SOAP/WSDL/UDDI `[2회 Q5]`, XML/JSON/HTML `[6회 Q10]`
- AJAX(비동기 부분 갱신) / 반응형 웹 `[1회 Q16]`
- 크롤링(수집) / 스크래핑(추출) `[16회 Q19]`
- 웹 인프라: L4(부하분산)/WEB(정적)/WAS(동적)/JVM `[20회 Q12]`

### 미들웨어 / 통신 기법

- RPC(원격 함수, 동기) / MOM(메시지 큐, 비동기) `[10회 Q9]`
- IPC: 파이프(FIFO 단방향) / 소켓(네트워크 양방향) `[17회 Q9]`
- EAI: P2P / Hub&Spoke / Message Bus `[15회 Q20]`

## 2. 정보보안 / 암호

### 정보보안 3요소 & 위험 관리

- 기밀성 / 무결성 / **가용성**(Land Attack·DoS 침해) `[3회 Q7]`
- 식별 → 인증(비번) → 인가(권한) / 부인방지 `[2회 Q4]`
- ⭐ **위협(Threat) / 취약점(Vulnerability) / 위험(Risk)** `[16회 Q14]`

### ⭐ 공격 유형 총정리

| 공격 | 특징 | 회차 |
| -- | -- | -- |
| Land Attack | SIP=DIP, 자기응답 DoS | `[3회 Q7]` |
| 살라미 / 스턱스넷 / 제로데이 | 소액 / 기반시설 웜 / 패치 전 | `[1회 Q7]` |
| DNS 스푸핑 / 파밍 | DNS 위조 / 캐시 오염 | `[7회 Q12]` |
| DRDoS | 반사 서버 증폭 | `[8회 Q13]` |
| 중간자(MITM) | 통신 가로채 복호화·재암호화 | `[9회 Q12]` |
| XSS / CSRF | 스크립트 주입 / 위조 요청 | `[5회 Q13]` |
| 딥페이크 | AI 조작 콘텐츠 | `[11회 Q11]` |
| 이블 트윈 | 가짜 와이파이 | `[14회 Q11]` |
| 타이포스쿼팅 / ARP스푸핑 / 스니핑 / IP스푸핑 | 오타 URL / MAC 위조 / 도청 / IP 도용 | `[13회 Q20]` |
| 능동(변조·재전송·위장·DoS) vs 수동(도청·분석·감시) | 공격 분류 | `[10회 Q7]` |

### ⭐ 암호 알고리즘 총정리

| 구분 | 알고리즘 | 특징 | 회차 |
| -- | ---- | -- | -- |
| 대칭키 | DES/AES/SEED/ARIA/LEA | 동일키, 빠름 | `[6회 Q19]` `[18회 Q16]` |
| 대칭 블록 | DES(64/56), AES(128) | NIST 표준 | `[15회 Q19]` |
| 대칭 스트림 | RC4 | SSL/TLS, 취약 | `[15회 Q19]` |
| 비대칭키 | RSA(소인수분해) | 키교환/서명 | `[6회 Q19]` |
| 비대칭 | ElGamal/ECC(타원곡선) | 이산대수 | `[14회 Q9]` `[15회 Q19]` |
| 해시 | MD5(128)/SHA | 무결성, 일방향 | `[15회 Q19]` `[13회 Q17]` |
| 키 교환 | Diffie-Hellman | 이산대수 키교환 | `[17회 Q12]` |

- 국산 대칭키: SEED(1999)/ARIA(2004)/LEA(2013, IoT 경량) `[18회 Q16]`
- ⭐ 공개키 용도: **암호화=상대 공개키 / 전자서명=내 개인키** `[10회 Q1]`
- 비번 보호: 솔팅(난수 추가) / 키 스트레칭(반복 해시) `[11회 Q1]`
- DB 암호화: API(앱 수준) / TDE(엔진 투명) `[10회 Q5]`
- 스테가노그래피(존재 은닉) vs 크립토그래피(내용 은닉) `[15회 Q13]`

### 보안 프로토콜 / 모델 / 시스템

- SSL → TLS(보완) `[17회 Q12]`, SET(전자상거래) `[15회 Q3]`
- IPSec: AH(인증) / ESP(암호화+인증) `[11회 Q18]`
- 무선: WEP(취약) → WPA(TKIP/AES) `[20회 Q14]`
- 이메일 보안: S/MIME / PGP `[16회 Q20]`
- ⭐ 무결성 모델: 비바(Biba)/클락-윌슨/만리장성 `[10회 Q3]`
- ⭐ 접근제어 모델: **벨-라파둘라(기밀, NRU/NWD)** vs **비바(무결, NRD/NWU)** `[15회 Q16]`
- 접근제어: MAC(강제)/DAC(소유자)/RBAC(역할) `[2회 Q18]`
- IDS(탐지)/IPS(차단) `[4회 Q17]`, IDS 방식: 이상 탐지 / 오용 탐지 `[14회 Q16]`
- 제로 트러스트(Never Trust) `[8회 Q8]`
- 보안 관리: SIEM(통합)/SOAR(자동화)/ESM(전사) `[13회 Q9]`
- 방화벽: 스크리닝 라우터 / 배스천 호스트 / 듀얼 홈드 `[20회 Q3]`
- 시큐어 코딩 / OWASP(웹 취약점 Top10) `[19회 Q15]`
- 콘텐츠 보호: DRM/워터마킹/위변조 방지 `[11회 Q4]`
- 개인정보/보안 인증: PIMS / CC(공통평가기준) `[18회 Q5]`

## 4. 소프트웨어 공학 / 설계

### 개발 모델 / 방법론

- 폭포수(순차) `[3회 Q9]`, 나선형(계획→위험분석→개발→평가) `[8회 Q14]`
- 애자일 스크럼: 스프린트/스프린트 백로그/데일리 스크럼/PO `[16회 Q17]`
- XP: 짝 프로그래밍/TDD/리팩토링/고객 상주/CI `[17회 Q7]`

### 요구사항 / 검토

- 요구공학: 도출 → 분석 → 명세 → 확인 `[13회 Q18]`
- 요구 도출: 인터뷰(1인)/브레인스토밍(집단) `[9회 Q4]`
- 검토: 워크스루(비공식)/인스펙션(공식)/리팩토링(구조) `[1회 Q11]`
- 협약 설계: 선행/결과/불변 조건 `[14회 Q4]`

### ⭐ 응집도 / 결합도

**응집도**(높을수록 좋음) `[6회 Q11]`
```
우연적 < 논리적 < 시간적 < 절차적 < 통신적 < 순차적 < 기능적
```
**결합도**(낮을수록 좋음) `[4회 Q19]`
```
자료 < 스탬프 < 제어 < 외부 < 공통 < 내용
```
- 스탬프 결합도: 구조체 통째 전달 일부만 사용 `[12회 Q19]`
- 팬인(호출받는 수)/팬아웃(호출하는 수) `[19회 Q16]`

### ⭐ 테스트

**커버리지 위계** `[7회 Q2]`
```
문장 ⊂ 결정(분기) ⊂ 조건/결정 ⊂ MC/DC ⊂ 다중조건
(주의) 조건 커버리지는 결정을 보장 못 함
```
- MC/DC(각 조건 독립 영향) `[4회 Q13]`

## 8. 계산 문제 공식 모음

### ⭐ 서브넷 `[1회 Q20]` `[3회 Q6]` `[9회 Q20]`

```
사용 가능 호스트 = 2^h − 2  (h=호스트 비트)
서브넷 개수     = 2^s        (s=서브넷 비트)
서브넷 마스크    = 32 − h 비트
예: /24에서 호스트 40+ → 2^6−2=62 → h=6 → /26, 서브넷 4개
```

### CPM(임계경로) `[3회 Q11]`

```
ES = 선행 EF 최댓값,  EF = ES + 기간
LF = 후행 LS 최솟값(없으면 종료일),  LS = LF − 기간
Slack = LS − ES,  임계경로 = Slack 0(최장 경로)
```

### 스케줄링 평균 반환시간 `[1회 Q12]` `[5회 Q20]` `[11회 Q16]`

```
반환시간 = 완료 − 도착,  평균 = Σ/n
RR: 퀀텀마다 재삽입(도착 프로세스 먼저)
SJF(짧은 것 먼저) = 평균 최소
```

### LOC / 순환 복잡도

```
인월(MM) = LOC / 생산성,  개발비 = MM × 인건비  [6회 Q17]
순환 복잡도 V(G) = 판단 노드 수 + 1 = E − N + 2  [13회 Q10]
```

### 가상주소 / 비트 `[2회 Q20]`

```
오프셋 비트 = log2(페이지 크기),  VPN 비트 = 전체 − 오프셋
```

### 페이지 부재 `[4회 Q9]`

```
LRU: 가장 오래 안 쓴 것 교체
LFU: 가장 적게 쓴 것(동률 FIFO)
OPT: 앞으로 가장 오래 안 쓸 것(이상적)
```

### 테스트 커버리지 위계

```
문장 ⊂ 분기(결정) ⊂ 조건/결정 ⊂ MC/DC ⊂ 다중조건
※ 조건 커버리지 ⊄ 결정 커버리지 (반례 존재)
```

### 정규화 단계

```
비정규 → 1NF(원자값) → 2NF(부분종속 제거) → 3NF(이행종속 제거)
→ BCNF(모든 결정자가 후보키) → 4NF(다치종속) → 5NF(조인종속)
```

### 교착상태 4조건 & 처리

```
4조건: 상호배제 · 점유대기 · 비선점 · 환형대기 (모두 성립해야)
처리: 예방(조건 차단) / 회피(은행가) / 탐지 / 회복
```

### 보안 접근제어 모델 vs 정책

```
정책(누가 결정): MAC(시스템 강제) / DAC(소유자) / RBAC(역할)
모델(무엇 보장): 벨-라파둘라(기밀, NRU/NWD) / 비바(무결, NRD/NWU) / 클락-윌슨(트랜잭션)
```

### 암호 큰 분류

```
대칭키(빠름, 키 교환 문제): DES / AES / SEED / ARIA / LEA / RC4
비대칭키(키 관리 용이, 느림): RSA / ECC / ElGamal / Diffie-Hellman
해시(일방향, 무결성): MD5 / SHA
```

### GoF 디자인 패턴 분류

```
생성: 팩토리메서드, 추상팩토리, 빌더, 프로토타입, 싱글턴
구조: 어댑터, 브리지, 컴포지트, 데코레이터, 퍼사드, 플라이웨이트, 프록시
행위: 옵저버, 전략, 템플릿메서드, 커맨드, 상태, 책임연쇄, 반복자, 중재자, 메멘토, 방문자, 인터프리터
```

### OSI 7계층 + 장비 + 프로토콜 (한 줄 요약)

```
1물리(리피터/허브) 2데이터링크(스위치/브리지, MAC, HDLC) 3네트워크(라우터, IP/ICMP/IGMP)
4전송(TCP/UDP) 5세션 6표현(암호화/압축) 7응용(HTTP/FTP/SMTP/SNMP/DHCP)
```
